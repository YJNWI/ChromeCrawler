# 🌐 Chrome Web Crawler
Fast multithreaded web crawler that explores the web beyond a single domain, prioritizing meaningful content and filtering noise.

## Features

- 🚀 Multithreaded crawling (fast)
- 🌍 Cross-domain exploration
- 🧠 Link prioritization (blogs, news, research, docs)
- 🧹 Noise filtering (login, tracking, social media, etc.)
- ⚖️ Domain limiting (avoid getting stuck in one site)
- 📄 Markdown export

## Usage

```bash
python crawler_links.py
